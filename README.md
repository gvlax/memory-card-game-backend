## memory-card-game-backend
An implementation of a REST API server for a memory card game frontend.

##### Instalation

The backend was develop on Node.js v7.10.
Check your Node.js version:

```
$ node -v
```
of using nvm:

```
$ nvm ls
```

##### Getting started

```
$ npm run dev
```
